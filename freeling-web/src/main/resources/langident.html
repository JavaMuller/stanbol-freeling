Language Identification Service

This stateless interface allows the caller to identify the language for submitted content.

This service is intended to be used in combination with the Stanbol Enhancer RESTful Language Identification Engine (STANBOL-895). To use this service with the RESTful Language Identification Engine just configure this URI as Service URI to an Engine instance.

RESTful interface

The content to analyse should be sent in a POST request with the Content-type header set to 'text/plain'. The response will be a JSON array containing information about the identified languages as specified by STANBOL-894

curl -X POST -H "Content-type: text/plain" \
     --data "The Freeling lanugage identification service can detect the \
         language of the parsed text." http://{host}:{port}/langident

The Response will look like

 [{
     "lang":"en",
     "prob":0.9688454784176675
 },{
     "lang":"fr",
     "prob":0.6318333375102646
 },{
     "lang":"es",
     "prob":0.5973599954559282
 }]
